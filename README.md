## Development

You need `node` and `npm` installed.

```
  git clone git@github.com:tagdesklimas/tdk-game.git
  cd tdk-game
  npm start
```

Go to `localhost:4444`.



## Production build

```
  npm run build
```

Analyze bundle composition (will open browser window):

```
  ANALYZE_BUNDLE=true npm run build
```
